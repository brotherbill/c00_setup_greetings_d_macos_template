# c00_setup_greetings_d_macos

Minimal D project with working macOS Apple Silicon build and F5 debug setup in VS Code.
Also serves as the setup repository for creating new D projects on macOS.

## Documents

| Document | Purpose |
|---|---|
| [setup_D_environment_for_MacOS.md](setup_D_environment_for_MacOS.md) | Full environment setup: macOS, XCode, VS Code, LDC, extensions, first build and F5 debug |
| [install_serve_d_macos.md](install_serve_d_macos.md) | Install the D language server (serve-d) for autocomplete and IDE features in VS Code |
| [setup_new_d_project_global.md](setup_new_d_project_global.md) | Install `new_d_project` globally so it can be called from any directory |
| [new_d_project.md](new_d_project.md) | Create a new D project from the template |
| [clean.md](clean.md) | Remove build artifacts from a D project |

## Quick Start

For a fresh macOS Apple Silicon machine, follow the documents in this order:

1. [setup_D_environment_for_MacOS.md](setup_D_environment_for_MacOS.md)
2. [install_serve_d_macos.md](install_serve_d_macos.md)
3. [setup_new_d_project_global.md](setup_new_d_project_global.md)
4. [new_d_project.md](new_d_project.md)
5. [clean.md](clean.md)
